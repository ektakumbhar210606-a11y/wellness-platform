
```

```

## Responsive Behavior

### Desktop (>1024px)
- Full table visible
- All columns displayed
- Standard pagination

### Tablet (768px - 1024px)
- Horizontal scroll enabled
- Fixed first column (Therapist Name)
- Scrollable remaining columns

### Mobile (<768px)
- Compact table view
- Larger touch targets
- Simplified pagination
- Swipe gestures for horizontal scroll
